О себе
Додов Шохкарим Frontend - разработчик

Frontend - разработчик. Верстаю адаптивные сайты в точности по макету, подключаю стороные скрипты, а также разрабатываю свои. После выпонения своей работы тестирую свою верстку на кросбраузерность, адаптивность, валидность, скорость загрузки. Владею современными технологиями HTML, CSS(Flexbox, Grid), Bootstrap, JavaScript, JQuery, LESS, SASS, PUG, BEM-naming, Git, Webpack, Gulp, работа в терминале. Постояно развиваюсь, совершенствую свои навыки, изучаю что-то новое для себя и очень люблю свою работу.

Учёба


HTML
Изучил язык гипертекстовой размтетки текста - HTML. Научился делать разметку веб-страницы, применять теги и атрибуты на сайте.
СSS
Изучил таблицу каскадных стилей СSS. Научился стилизовать элементы на сайте с помощью селекторов, атрибутов и классов тегов.
JavaScript
Изучил язык прогарамировония - JavaScript для создания интерактива на сайте.
JQuery
Изучил библиотеку JavaScript - JQuery для быстрого написания кода и подключение стороных решеный (слайдера, модальные окна, табы и т.д.).
Методология Бем
Изучил методологию верстки BEM-naming. Научился прминять в верстке методологию БЕМ(именование классов) для удобочитаемого кода.
Bootstrap
Изучил фреймворк CSS - Bootstrap. Научился применять готовую сетку Bootstrap в своих проектах.
Система контроля версии - GIT
Изучил систему контроля версии Git для работы, с удаленными репозиторями. Научился пушить, пулить, делать коммиты, создавать ветки и т.д.
Aвтоматизации сборки
Научился применять различные автоматизации сборки для ускоренной работы (Gulp, Webpack), а также препроцессоры Less/Sass, Pug.
Верстка
Изучил верстку страниц сайта по PSD макету. Научился верстать адаптивно и кроссбраузерно с применением медиа запросов.
SVG
Изучил векторную гафику SVG. Научился добовлять векторную графику на сайт, а также стилизовать.
